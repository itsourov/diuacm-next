# Advanced Programming Camp Topics - Spring 2025

Welcome to the Advanced Programming Camp! This comprehensive program is designed to take you through various aspects of competitive programming, algorithms, and data structures. Below is the detailed curriculum for each class.

## Class Schedule Overview

### Class 00: Introduction
**Trainer:** Saiful Islam (Assistant Professor, CSE)
- About Advanced Camp
- Introduction to Competitive Programming

### Class 01: Programming Basics
**Trainers:** Kazi Amir Hamza & Kh Sadman Sakib
- Data types and I/O formatting
- File I/O
- Understanding OJ verdicts
- Time Complexity Analysis
- Bug Tracing Techniques
- Stress Testing
- Problem Solving Strategies

### Class 02: Common Techniques & Greedy Algorithms
**Trainers:** Md. Sijan & Kazi Amir Hamza
- Prefix Sum
- Difference Array
- Two Pointers Technique
- Sliding Window
- Classic Greedy Problems
- Kadane's Algorithm

### Class 03: Recursion & Complete Search
**Trainers:** Sourov Biswas & Whahiduzzaman Payel
- Function Overview
- Parameters and Return Types with Examples
- Loop vs Recursion
- Recursion Exercises
- Backtracking

### Class 04: Introduction to C++ & Data Structures
**Trainers:** Uodoy Hossain & Piyash Bashak
- STL Containers
- Advanced Data Structures:
  - Min Stack
  - Sparse Table
  - DSU (Disjoint Set Union)

### Class 05: Searching and Sorting
**Trainers:** Uodoy Hossain & Whahiduzzaman Payel
- Linear Search
- Binary Search (including binary search on answer)
- Ternary Search
- Sorting Algorithms:
  - O(N²) Sorting Algorithms
  - O(N log N) Sorting Algorithms
  - Merge Sort and Tree Sort
  - STL Sort
  - Custom Sorting Functions

### Class 06: Number Theory (Full Day)
**Trainers:** Kazi Amir Hamza & Saimum Islam Hamza
- Base Conversion
- GCD & LCM (Log₂N implementation)
- Modular Arithmetic
- Prime Numbers:
  - Prime Checking
  - Factorization
  - Sieve of Eratosthenes
  - Segmented Sieve
- Modular Inverse
- Binary Exponentiation
- Euler's Totient Function (ETF)

### Class 07: Graph Theory
**Trainers:** Piyash Bashak & Saimum Islam Hamza
- Graph Representation
- Graph Traversal:
  - BFS (Breadth-First Search)
  - DFS (Depth-First Search)
- Shortest Path Using BFS
- Multi-Source BFS

### Class 08: Advanced Data Structures
**Trainers:** Uodoy Hossain & Kh Sadman Sakib
- Segment Tree
- Policy Based Data Structures (PBDS)
- Unordered_map
- Custom Hash Functions
- Randomization Techniques

### Class 09: Bit Manipulation & Combinatorics
**Trainers:** MD. Masum Billah & Kh Sadman Sakib
- Bit Manipulation:
  - Basic Operations
  - Tips and Tricks
- Combinatorics:
  - Permutations
  - Combinations
  - nCr Calculations

### Class 10: Dynamic Programming
**Trainers:** MD. Masum Billah & Kh Sadman Sakib
- Understanding States and Transitions
- Recursive Dynamic Programming
- Converting Recursive to Iterative Solutions
- State Optimization Techniques

### Class 11: Strings & Geometry
**Trainers:** Masum Billah & Saimum Islam Hamza
- String Hashing
- Geometry Fundamentals:
  - Basic Geometry
  - Trigonometry
  - Coordinate Geometry
  - Vectors
- Polygon Algorithms

### Class 12: Advanced Graph Theory
**Trainers:** Piyash Bashak & Saimum Islam Hamza
- Dijkstra's Algorithm
- Topological Sorting
- Binary Lifting
- Lowest Common Ancestor (LCA)
